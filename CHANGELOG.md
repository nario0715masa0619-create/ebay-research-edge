# CHANGELOG - eBay Research Edge

すべての重要な変更は このファイルに記録されます。

フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/) に基づいています。
バージョニングは [Semantic Versioning](https://semver.org/ja/) に従っています。

## [15.2] - 2026-05-08

### 追加
- **Yahoo!オークション落札履歴統合**：落札相場から最小・最大・中央値を自動計算
- **AI 画像判定強化**：GPT-4o Vision で eBay 画像と国内商品画像を比較
- **レプリカ・観賞用品検知**：商品説明から「レプリカ」「観賞用」「展示用」キーワードを検出し、一致度を強制 0% 設定
- **仕入れ値編集機能**：ダッシュボード上でテキストボックスで仕入れ値を手動入力可能に
- **リアルタイム利益計算**：仕入れ値変更時に利益・ROI を自動再計算
- **Google Sheets GAS 連携**：Web App を経由した Google Sheets への自動保存機能

### 変更
- **Yahoo!オークション統計**：平均値から中央値に変更（外れ値の影響を減少）
- **メルカリ DOM セレクタ更新**：最新の HTML 構造に対応（NoTitle バグ修正）
- **キーワード抽出ロジック改善**：カード番号・レアリティを優先抽出、正規表現の単語境界を厳密化
- **ハードオフ解析簡素化**：リンク・値抽出ロジックをシンプル化、除外キーワードチェック強化

### 修正
- **Python regex SyntaxWarning**：無効なエスケープシーケンスを修正
- **スクレイピング API キー漏洩**：Git 履歴から削除、新キー生成対応
- **非同期処理の安定性**：asyncio タイムアウト設定を厳密化

### 削除
- **不要なスクレイピング API**：SCRAPEDO_API_KEY, DIFFBOT_API_TOKEN, OCTOPARSE_API_KEY を削除
