# eBay-Research-Edge Acceptance Criteria

## 1. 機能要件
- キーワード検索ができる
- eBay Sold データを取得できる
- 国内価格データを取得できる
- 30日 / 90日集計ができる
- 価格中央値 / 平均値 / 最安値 / 最高値が表示される
- 概算利益 / 粗利率が表示される
- STRが表示される
- LIST候補 / 保留 / 見送りの判定が表示される
- CSV出力ができる

## 2. 非機能要件
- 処理失敗時にエラー内容が分かる
- 生データを保持できる
- 正規化処理を再実行できる
- スコア計算ロジックを変更しやすい
- コード構造が責務分離されている

## 3. 完成判定
以下をすべて満たしたら MVP完了とみなす。

- 1ジャンルで動作する
- eBay と国内1サイトの比較ができる
- LIST候補抽出が動く
- CSV出力できる
- README と docs が更新されている
